## Installing RecordarFotos-UI

To use RecordarFotos-UI components, all you need to do is install the
`RecordarFotos-UI` package and its peer dependencies:

```sh
$ yarn add recordarFotos-ui

# or

$ npm i recordarFotos-ui
```
